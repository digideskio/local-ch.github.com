Frontend Web Developer

As a frontend engineer you will focus on building web apps and creating a great user experience. We base our development on modern frameworks such as AngularJS and Ruby on Rails and combine a mix of agile practices, pair programming and code reviews to improve the process of working in a team. We take testing and automation seriously. As a small team, we believe we have constructed an environment that allows us to continue to adapt and develop cutting edge features.

Your responsibility is end-to-end, from planning, implementation, testing to maintaining your code after deployment. You are part of a solid team of engineers, working with product owners, specialists and other teams. New features are implemented and deployed in short iterations and your contributions and ideas are always welcomed.

While we are running one of the biggest web sites in Switzerland, you get a chance to have a positive impact on millions of users within your first day.

Roles and Responsibilities

  • Delight users with great products.
  • Develop web and responsive apps for broadscreen and tablets.
  • Assist product managers in understanding audience (A/B tests and data analysis).
  • Transform mockups/designs into cross browser web apps.
  • Help improve the engineering tool chain and process.

Desired Skills & Experience

  • Ideally 3+ years professional experience in web software development.
  • Can demonstrate skills in developing rich web applications and user interaction.
  • HTML and CSS (HAML/SCSS a plus).
  • Strong Javascript skills.
  • Experience with JS frameworks like AngularJS, Backbone.js, Ember.js or Meteor.
  • Understanding of cross-browser development, testing and bug fixing.
  • Experience with Ruby on Rails or similar server side MVC framework.
  • Excited by challenging problems, addicted to shipping features to production.
  • Advanced understanding of HTTP protocol.
  • Understanding principles of REST architectures.
  • Experience with at least one other programming language as PHP, Python, Java, C++ ...
  • Comfortable at a Unix command-prompt.
  • Fluent in English. German/French is a plus.

Apply Today!

Questions?
