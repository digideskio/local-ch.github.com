Frontend Web Developer

As a frontend engineer you will focus on building web apps and creating a great user experience. We develop applications in Ruby on Rails and Javascript. Our team works with a mix of agile practices, pair programming and code reviews to continuously improve our code and ourselves. We take testing and automation seriously. At local.ch, we believe we have constructed an environment that allows us to continue to adapt and develop cutting edge features.

Your responsibility is end-to-end, from planning, implementation, testing to maintaining your code after deployment. You are part of an experienced team of engineers, working with product owners, specialists and other departments. New features are implemented and deployed in short iterations and your contributions and ideas are always valued.

Since we are running one of the biggest web sites in Switzerland, you get a chance to have a positive impact on millions of users within your first day.

Roles and Responsibilities

  • Delight users with great products.
  • Develop web and responsive apps for browsers and tablets.
  • Assist product managers in understanding audience (A/B tests and data analysis).
  • Transform mockups/designs into cross browser web applications.
  • Help us improve the engineering tool chain and process.

Desired Skills & Experience

  • Ideally 3+ years professional experience in web software development.
  • Can demonstrate skills in developing rich web applications and user interaction.
  • HTML and CSS (HAML/SCSS a plus).
  • Strong Javascript skills.
  • Experience with JS frameworks like AngularJS, React, Backbone.js, Ember.js or Meteor.
  • Understanding of cross-browser development, testing and bug fixing.
  • Experience with Ruby on Rails or similar server side MVC framework.
  • Advanced understanding of HTTP protocol.
  • Understanding principles of REST architectures.
  • Experience with at least one other programming language as PHP, Python, Java, C++ ...
  • Comfortable at a Unix command-prompt.
  • Confidence with using Git
  • Good habits with writing tests (preferably in RSpec and Javascript)
  • Excited by challenging problems, addicted to shipping features to production.
  • Fluent in English. German/French is a plus.

Apply Today!

No cover letter required. Simply send your CV and a link to your GitHub account or code examples to hrz@local.ch.

Questions?
