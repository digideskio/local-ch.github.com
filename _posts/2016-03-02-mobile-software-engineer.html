Mobile Software Engineer

Overview

At local.ch we maintain and advance native applications for Android and iOS used by more than a million clients a month.. We’re looking for a Mobile Software Engineer to help us do exactly that.

As a Mobile Software Engineer you will focus on building apps and creating a great user experience. Our team works with agile practices, pair programming and code reviews to continuously improve our code and ourselves.

Your responsibility is end-to-end, from planning, implementation, testing to maintaining your code after deployment. You are part of an experienced team of engineers, working with product owners, specialists and other departments. New features are implemented and deployed in short iterations and your contributions and ideas are always valued.

Since we are running some of the biggest mobile applications in Switzerland, you get a chance to have a positive impact on millions of users within your first day.

Roles and Responsibilities:

  • Delight users with great application features.
  • Develop native mobile applications (Android and/or iOS) for smartphones and tablets.
  • Transform mockups/designs (that you helped shape) into working native applications
  • Help us improve the engineering tool chain and process

Desired Skills & Experience

  • Degree in Software Engineering and/or 3+ years professional experience in software development
  • Proficiency in native application development on iOS (Objective-C/Swift) and/or Android (Java); having published your own app in the App or Play Store is a plus
  • Experience in the following areas is a plus: Ruby on Rails, RESTful APIs, Web applications, Git and testing
  • Experience refactoring code and evolving architectures
  • Strong understanding of computer science fundamentals, including object-oriented design, data structures, algorithm design, problem solving, and complexity analysis
  • Fluent in English. German/French/Italian is a plus.

Place of work: Zürich, Switzerland
