Java Software Engineer

If you are looking for a fast paced, challenging work environment and to be part of bleeding edge backend development team - Local.ch could be the right place for you. We are looking for a smart, motivated Java developer who is eager to take on new challenges. If you are an innovative thinker who is ready to work hard with an energetic team, we would like to talk to you.

We are looking for an experienced backend developer with a flair for frontend design. Hidden behind www.local.ch we created a webbased information (GWT) system to provide access to internal tools to support, sales and administration teams. We need YOU to become the application advocate and take ownership of it's core aspects: user experience, consistency, availability and performance. Java is our choice to develop the service infrastructure which covers a variety topics like realtime data flow management and indexing, ad delivery and of course our main field of development: search.

Roles and Responsibilities:

  • Completely immersed in agile product development and engineering efforts, working closely with all the different teams (web-, mobile-, content team)
  • Identify and improve usability issues, proactive availability and performance monitoring
  • Be faced with demanding business request, challenge the requirements, find and implement elegant solutions
  • Work partially autonomously, partially as a bigger team
  • Quality assurance (Unit and integration testing)
  • Collective code ownership
  • Deliver on time
  • End-to-end responsibilities: from planning, implementation, testing to maintaining your code after deployment

Desired Skills & Experience

  • Very strong Java expert knowledge
  • Flair for application Sensible to frontend usability issues and a strong will to provide the best and fastest service.
  • Ideally 3+ years work experience as a java software engineer
  • Capability to work rapidly and efficiently in an unstructured environment
  • Ability to acquire information autonomously
  • Frameworks: Spring, GWT, RestEASY, Hibernate
  • Knowhow with at least a sub selection of the following storage technologies: Lucene, SOLR, MongoDB, SQL, PostgreSQL, Redis
  • Experience in developing web service: Rest and SOAP
  • Classical frontend skills desired: HTML, CSS, JS - analytical knowhow to measure and monitor the performance of an web application
  • Tools: Subversion/GIT, Eclipse and different Atlassian products
  • Comfortable working in a Unix environment
  • Experience in Agile Development (XP, Scrum, Kanban)
  • Fluent in English. German/French is an advantage

    Place of work: Zürich, Switzerland
